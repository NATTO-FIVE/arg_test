内部記録：OSI参照モデルの基本定義

Security Classification: PUBLIC_REFERENCE_DATA
SUBJECT: 開放型システム間相互接続の参照モデル
ID: ISO/IEC 7498-1

OSI参照モデルは、コンピュータネットワークの機能を抽象化し、7つの階層に分割して定義したものである。異なるメーカーの機器間での通信を可能にするための国際標準フレームワークとして機能する。

Layer Name Description
L7Application具体的な通信サービス（HTTP等）
L6Presentationデータの形式、圧縮、暗号化
L5Session通信の開始から終了までの手順管理
L4Transport通信の信頼性、エラー訂正、再送制御
L3Network経路選択、論理アドレス（IP等）の識別
L2Data Link隣接ノード間のデータ転送、物理アドレス
L1Physical電気信号、コネクタ形状等の物理的定義

【注記】

本モデルは理論上の標準であり、実稼働環境ではTCP/IPプロトコルスタックが主流として用いられている。 各階層は独立しており、下位層のサービスを利用して上位層へサービスを提供する構造となっている。
